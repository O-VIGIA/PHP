//数据库
数据库 ：mysql
数据表：{
	用户：root
	内容：news
}

//功能实现
1)    注册：用户可以自行注册，要求对数据进行判断是否为空且是否重复；md5加密；
2)    登录：通过 COOKIE 或 SESSION 记录登录状态；登录后有Welcom的用户名欢迎；
	 若没登陆直接访问则会有注册登陆链接;    
3)    表白：用户只有登陆后才能表白，游客状态只能查看别人表白；
	 登陆后点击个人表白则可以管理自己的表白，发表，删除；
4)    浏览： 在主页点击内容后能查看的表白内容；
5)    删除：登录的用户进入个人表白页面可以删除自己的表白记录；
	 删除成功后会有提示页面，五秒跳转，也可以自行单机跳转；
6)    分页：主页有分页效果，因为数据太少，所有每五条记录一页；
	 (可修改)
